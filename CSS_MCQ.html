Course Basic Quiz (17 Jan'23)

1. What is the correct CSS property for hiding an element?

  • visibility: hidden;
  • display: none;
  • hidden: true;
  • invisible: yes;

Ans: display: none;

2. What is the correct CSS property for setting the text color of an element?

  • text-color
  • color
  • font-color
  • text-style

Ans: color

3. What is the correct CSS property for setting the space between elements?

  • padding
  • margin
  • spacing
  • distance

Ans: margin

CSS QUIZ (24 Jan'23)

1. What does CSS stand for?

  • Color style sheets
  • Complete style sheets
  • Cascading style sheets
  • Combine style sheets

Ans: Cascading style sheets

2. How many ways are there to bring CSS into an HTML file?

  • 2
  • 1
  • 3
  • 4

Ans: 3

3. To write inline CSS we use ______ tag?

  • Class
  • Style
  • Id
  • onstyle

Ans: Style

4. How do you import an external style sheet?

  • Import tag
  • Link tag
  • Script tag
  • Style tag

Ans: Link tag

5. What file extension should we give to a css file?

  • .style
  • .css
  • .stylesheet
  • .csssheet

Ans: .css
